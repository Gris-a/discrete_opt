# Set Cover

## Сборка
```bash
./build
```

## Тестирование
Запуск на оценку:
```bash
./set_cover
```

Запуск своего теста:
```bash
./set_cover <filename> \ # путь к тесту
            <bound1>   \ # порог на 3
            <bound2>   \ # порог на 5
```


## Отчёт
Изначально так как не известно, насколько большие пороги в условии, решил привести хоть какое-нибудь решение, поэтому взял жадный алгоритм (set_cover_greedy), он работает быстро и даёт не самое плохое решение:
```
sc_157_0    3/5
sc_330_0    0/5
sc_1000_11  3/5
sc_5000_1   3/5
sc_10000_2  3/5
sc_10000_5  3/5
```
