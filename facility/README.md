
## Отчёт
### Greedy
Для каждого клиента выбирается допустимый объект с минимальной суммы расстояния до клиента и стоимости открытия (если объект ещё закрыт). После назначения клиента объект помечается как открытый (если это было нужно) и его использованная ёмкость увеличивается.

| тест | значение | порог |
|:----:|:--------:|:-----:|
| fl_25_2   | 3269821.3   | 5 / 5 |
| fl_100_1  | 47302882.4  | 0 / 5 |
| fl_200_7  | 7765704.5   | 0 / 5 |
| fl_500_7  | 83975185.4  | 0 / 5 |
| fl_1000_2 | 12076465.4  | 0 / 5 |
| fl_2000_2 | 12479411.0  | 0 / 5 |
| Итого     |             | 5 / 30 |
